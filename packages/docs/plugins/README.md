All plugins operate on a tree generated by browse documentation

The tree takes the following form

```js
{
  [scope]: {
    description: [descriptionText],
    [rules]: {
      [ruleName]: {
        help: [helpText],
        description: [descriptionText],
        params: {
          name,
          description,
          type
        },
        rtn: {
          description,
          type
        }
        notes: [notesText],
        example: [exampleText]
      }
    },
    config: {
      [variableName]: {
        type,
        description
      }
    }
  }
}
```
